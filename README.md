<h1>Website Collection</h1>
 
This repository includes 5 websites:
About me: ["https://cci.arts.ac.uk/~elowen/AboutMe/"]
An introductory website utalising GSAP's Javascript animation library.

Day in the Life: ["https://cci.arts.ac.uk/~elowen/DayintheLife/"]
Come back to this website at different times of the day to see what they are up to at that moment! 

Wishing Well: ["https://cci.arts.ac.uk/~elowen/WishingWell1/"]
Flip a coin into the wishing well to generate a new fortune.

Who Wished me a Happy Birthday: ["https://cci.arts.ac.uk/~elowen/WhoWishedMeAHappyBirthday/"]
A self made API database that includes the characters names, text message screenshot accompanied with the text transcript, their own birth date, and a link leading to a website showing who they share their birthday with.

5.My Lovely Objects: ["https://cci.arts.ac.uk/~elowen/MyLovelyObjects/"]
A ThreeJS project showing lovely objects around my flat.
