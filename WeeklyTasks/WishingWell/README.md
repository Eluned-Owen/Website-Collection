Week four - A Website that is a Wishing Well

A website where you can toss a coin into a well where it will then bestow upon you a fortune.

Requirements:
-VS Code or any IDE of choice to run the HTML, CSS and JavaScript file
or Access to the website at ["https://cci.arts.ac.uk/~elowen/WishingWell1/"]
-Access to the internet is needed for GSAP animations

How to use:
-Download the code from the Github ["https://git.arts.ac.uk/23008862/Web-Development-Weekly-Tasks"]
-Open the file in the preffered IDE 
-Click the coin
-Watch as the well gives you a fortune

Inspired by an idea found at ["https://www.are.na/readymag/websites-that-should-have-existed-if-they-didn-t-but-they-do"] I decided to create a website that acts as a wishing well. I designed the background and well by hand, keeping to a dreamy purple theme to emphasise the feeling of mysetry and mistique from the unknown fortune. I utilised grids in order to get the elements to lay in the middle. Futhermore I intially used the :hover CSS feature to animate the coin to going into the well. I struggled greatly with displaying alerts seconds after the :hover was initiated, because it had trouble detecting when the condition was met. To solve this issue I decided to create the animation using GSAP and replaced the hovering with a click.

To improve:
I would add a reset button for the animation to go back to the start, allowing another coin toss. In addition to this I would like to add my own alerts instead of relying on the inbuilt browser's alerts as this would allow me to customise them as I would like.
