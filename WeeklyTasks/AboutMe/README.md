Week One - About Me

A website where you can learn more about me!

Requirements:
-VS Code or any IDE of choice to run the HTML, CSS and JavaScript file
or Access to the website at ["https://cci.arts.ac.uk/~elowen/AboutMe/"]
-Internet connection for GSAP compatibility 

How to use:
-Download the code from the Github ["https://git.arts.ac.uk/23008862/Web-Development-Weekly-Tasks"]
-Open the file in the preffered IDE 
-Ensure WIFI access to view GSAP animation

For my first week, I decided to create a modern/cyber styled website utilising GSAP's scrolling animtaion functionalities. This website goes over basic information about me, my likes and fun facts about myself. In addition to GSAP, I imported the Orbitron font found here ["https://fonts.google.com/specimen/Orbitron"] to give the document a futeristic sleek feel.
