Week 5- Day in a life

A website that changes throughout the day, showing what I'd typically be doing throughout the day.

Requirements:
-VS Code or any IDE of choice to run the HTML, CSS and JavaScript file
or Access to the website at ["https://cci.arts.ac.uk/~elowen/DayintheLife/"]

How to use:
-Download the code from the Github ["https://git.arts.ac.uk/23008862/Web-Development-Weekly-Tasks"]
-Open the file in the preferred IDE 
-Come back at any time through the day to see what I am doing!

I created 4 different pixelated illustrations for night, morning, noon and evening activities. This required me to use functions such as Date() and getHours() to process realtime time data.
